# GOBLOG

GoBlog is a blog-template for Golang. You will just edit JSON file and it going to be ready for you.

## Installation


```bash
git clone https://github.com/penguinuxx/GoBlog.git
```

## Usage
Editing after the JSON files 
```bash
cd GoBlog/
go run main.go
```
